# alisveris_app

Designed, built and tested a fully functional shopping application with Flutter and Firebase.

• OOP

• Responsive UI

• ListView Builder

• State Management - Provider

• Firebase
  - Authentication
  - Firestore
  - Storage

Google Play Linki: https://play.google.com/store/apps/details?id=com.alisveris_app
